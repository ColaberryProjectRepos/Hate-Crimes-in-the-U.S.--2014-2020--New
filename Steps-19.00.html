Create a Hate Crimes by Location Chart 19



1. Copy and paste the Hate Crimes by Year | Month chart and paste on the canvas.
2. Drag the chart to an open space on the canvas
3. Replace the X-axis with Location_Name
4. In Format Visual, set as follows: 

General

  • Properties: Height-323, Width-389
  • Position: Horizontal-468, Vertical-394
  • Title-On, Hate Crimes by Location, Text Color-#FFFFFF
  • Change chart to scatter chart




Visual

      X Axis

  • Values: Font-Trebuchet, Size-9, Color #F8C464, Display Units- Auto, Decimal-Auto, Title-On, Font-DIN (10), Color- #F8C464

      Y-Axis

  • Values: Trebuchet, Size-9, Color #287E8F, Display Units- Auto, Decimal-Auto, Title-On, Font-DIN (10), Color- #287E8
  • Markers-On




Visual Cont'd

  • Legend-On, Position-Top left, Font-Trebuchet MS (8), Title-On, Title-Hate Location, Category label-On
  • Format Style-Gradient, Field; Hate Crimes YoY%, Lowest Value-#5398A5, Middle Value-#F8C464, Highest value-#E2713B
