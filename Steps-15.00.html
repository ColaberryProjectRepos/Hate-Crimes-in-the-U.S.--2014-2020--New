Add KPIs to the Dashboard 15


We will need to first create a shape on which we will place the KPI cards. 

Steps (Adding a shape)

1. In the Home tab, click on Insert

2. Click on Shapes and select Rectangle

3. In Format, select style and format as follows:

  • Color-#FAFEFF; Border-On; Color-#CCCCCC; Width-1px; Text-Off; Shadow-Off

4. In Format, go to General and format

  • Properties: Height-108; Width-223
  • Position: Horizontal-0; Vertical-70
  • Title-Off
  • Effects: Background-On; Transparency-0%
  • Visual border-On; Color-#CCCCCC, Rounded corners-0px; Shadow-Off



5. Copy and paste the shape twice for the other two KPIs. See where to place the shapes in the image below 

Steps (Adding Hate Crime KPIs)
1. In Visualizations, select Card and place it on the shape. 
2. Drag and drop Hate Crime measure onto the card.
3. In Format, and set as follows:

Visual

  • Callout Value: Font-DIN; Size 32; Color #E2713B
  • Display units:-Auto; Value Decimal Places-1
  • Category label-Off

General

  • Properties: Height-108; Width-223
  • Position: Horizontal-0; Vertical-70
  • Title: On and enter Hate Crimes | YoY% in the Title box; Font: Trebuchet MS; Size-14, Text color-#F8FAFC; Background color-On; Color-#7EB2BC
  • Effects: Background-Off; Visual Border-On; Color #CCCCCC




Steps (Adding Hate Crimes YoY%)
1. In Visualizations, select Card and place it on the shape. 
2. Drag and drop Hate Crime YoY% measure onto the card.
3. In Format, and set as follows:

Visual

  • Category label-Off
  • Callout Value: Font-DIN; Size 20; Color #CB6636
  • Display units: Auto; Value Decimal Places-1

General

  • Properties: Height-158; Width-139
  • Position: Horizontal-0; Vertical-70
  • Title: Off
  • Visual Border-Off
  • Effects: Background-Off
  • Visual border-Off
