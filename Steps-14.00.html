Create Measures Table and DAX Measures 14


Steps: Create measures table and add Hate Crime measure

1. In the Home tab of the Desktop ribbon, click Enter Data. A screen will that allows you to manually enter data will pop up

2. In Name the table, type “All-Measures”

3. Right-click on All_Measures and select New Measure

4. In the Dax formula editor, enter the measure:  Hate Crimes = COUNTROWS('Hate Crimes in the US')




Repeat (3) above and enter the following measures. In the DAX formula box, copy and paste these measures, one at a time.

 

Hate Crimes YoY%

    a. Avg Total Hate Crimes = AVERAGEX(VALUES('Calendar'[Date].[Year]), [Hate Crimes])

 

    b. Hate Crimes YoY% = VAR PYHateCrime =

       CALCULATE([Avg Total Hate Crimes], DATEADD('Calendar'[Date],-1,YEAR))

       RETURN

       IF(ISBLANK(PYHateCrime),0,DIVIDE(([Avg Total Hate Crimes]-  

      PYHateCrime),PYHateCrime))

 

Violent Hate Crimes KPIS

    a. Violent Hate Crimes(VHC) = CALCULATE(COUNTROWS('Hate Crimes in the US'),

    'Hate Crimes in the US'[Crime_Type]="Violent Hate Crime")

 

    b. VHC Rate = DIVIDE([Violent Hate Crimes(VHC)],'All_Measures'[Hate Crimes])

 

    c. Avg VHC = AVERAGEX(VALUES('Calendar'[Date].[Year]), [Violent Hate Crimes(VHC)])

 

    d. VHC YoY% = VAR PYVhc = CALCULATE([Avg VHC],

       DATEADD('Calendar'[Date],-1,YEAR))

       RETURN

       IF(ISBLANK(PYVhc),0,DIVIDE(([Avg VHC]-PYVhc),PYVhc)) 

 

Non-Violent Hate Crimes KPIs

     a) Non-Violent Hate Crimes(VHC) = CALCULATE(COUNTROWS('Hate Crimes in the

         US'), 'Hate Crimes in the US'[Crime_Type]="Non-Violent Hate Crime")

 

    b) Non-VHC Rate = DIVIDE([Non-Violent Hate Crime (N-VHC)],'All_Measures'[Hate Crimes])

 

    c) Avg N-VHC = AVERAGEX(VALUES('Calendar'[Date].[Year]), [Non-Violent Hate Crime (N-VHC)])

 

    d) VHC YoY% = VAR PYVhc = CALCULATE([Avg VHC],

        DATEADD('Calendar'[Date],-1,YEAR))

        RETURN

         IF(ISBLANK(PYVhc),0,DIVIDE(([Avg VHC]-PYVhc),PYVhc))
