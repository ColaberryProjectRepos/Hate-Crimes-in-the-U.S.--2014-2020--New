Create a Calendar table and Refresh Date table 10

Steps (Calendar table)


1. In the Model View on Power BI Desktop, click on New

2. Enter the DAX formula below, click on the check mark and save.


Calendar =

ADDCOLUMNS (

CALENDAR (Date(Year(Min('Hate Crimes in the US'[Incident_Date])),

Month(Min('Hate Crimes in the US'[Incident_Date]))

,DAY(Min('Hate Crimes in the US'[Incident_Date])))

, Date(Year(Max('Hate Crimes in the US'[Incident_Date]))

,Month(Max('Hate Crimes in the US'[Incident_Date]))

,DAY(Max('Hate Crimes in the US'[Incident_Date])))),

"Month number", FORMAT ( [Date], "MM" ),

"Month Year", FORMAT ( [Date], "MMMYYYY" ),

"Month", FORMAT ( [Date], "mmmm" ),

"Week Number", "Week-"&WEEKNUM( [Date] ),

"Week", WEEKNUM ( [Date]),

"Day Of Month", Month([DATE]) &"-"& FORMAT ( [Date], "dd" ),

"Year Quarter", FORMAT ( [Date], "YYYY" ) & "/Q" & FORMAT ( [Date], "Q" ),

"Year", FORMAT ( [Date], "YYYY" ))





Steps (Refresh Date table)



1. Go to power query editor under transform data, then click on New source then blank query.

2. Then write the query = DateTime.LocalNow() in the Formular bar

3. Convert it to a table and rename it. A new column will be created. Rename the column DateTime

4. Duplicate the column and rename it Date. Go to Date in the tool bar and format the date to Date/Date Only

5. Duplicate the first column a second time and rename the new column Time. Go to Time in the tool bar and change the format to time/ time only.
