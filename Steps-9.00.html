Steps: (Creating conditional columns for violent hate crimes (VHC), non-violent hate crimes (N-VHC) and damage to property)

1. Select Transform Data from the Home tab

2. Under power query Editor click on the Add Column tab

3. Select Conditional Column

4. Enter Crime_Type in the New Column Name box

5. In the IF statement box, select the column name, an operator (in this case "contains) and the value and output as shown in the images below.

6. Click on "Add Clause" to add the next set of values.



7. Repeat step 6 above and add all the values as shown below
8. Once you finish, Press OK then Click and Apply
