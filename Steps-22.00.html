Create KPI indicators 22

Now, we can create the KPI indicators.



1. From Visualization, drag and drop and KPI card onto the canvas. Drag and drop the Icon Hate Crimes YoY% measure in Field

2. Go to Format, select Visual and format as follows:

Callout: Font-Trebuchet MS, Size-20

3. Under color, click fx and for Format Style, select Field Value

4. Under what the field should be based on, select Icon Hate Crime YoY color

5. Turn the Category off.

6. Format the Callout under Visual as follows: Font-Trebuchet MS; Size-20

7. Under Color, click fx  and a dialogue will open

8. Under Format Style, select Field Value

9. Under the field it is based on, select Icon Hate Crimes YoY Color

10. Turn Category label off.

General

        Position: Horizontal-166; Vertical 106

        Title-Off

        Effects: Background-Off; Visual border-Off; Shadows-Off; Header Icons-Off, Tooltips-Off
