Create a Hate Crime YoY% Waterfall Chart 21


Waterfall Chart

First, we will need to create a table to hold the values for the Waterfall Chart as well as a Variance Measure

Steps
1. Go to Home and on Home ribbon, click on Enter Data

2. Name the first Column ID and the second one Measures Desc

3. In the Name Box, name the table Hate Crime Breakdown



4. Create a Measure to display the 3 performance indicators. 
5. Right click the Measures table
6. In the DAX box, enter the DAX formula below
7. Name the Measure-Hate Crime Trend

Display YoY% = 

SWITCH (

    SELECTEDVALUE (Hate Crime Breakdown[ID]),

    1, [Hate Crimes YoY%],

    2, [VHC YoY%],

    3, [N-VHC YoY%]

)



8. Click on an empty space on the canvas, and from the visualizations pane select Waterfall Chart.

9. Drag the Measures Desc column from Hate Crime Breakdown table to the Category.  Then drag the Hate Crime Trend Measure into the Y-axis. Drag the following onto the Tooltips field:

  • Hate Crimes
  • VHC Rate
  • N-VHC Rate
  • Hate Crimes YoY%
  • VHC YoY%
  • N-VHC YoY%



 Format the chart

9. In the Formatting Tab, format
General

Properties: Height-327, Width-302
Position: Horizontal-678, Vertical-70
Title On, Title-Hate Crime YoY%, Font-Trebuchet MS (14), Color-#FFFFFF, Background color-#7EB2BC
Effects: Background Off
Visual Border-On, Color-#CCCCCC



Format Visual

10. Go to Visual and format as follows: 

X-Axis

  • Values: Font-Trebuchet MS, Size-9, Color-1A1A1A, Display units-Auto, Title-Off

Y-Axis

  • Values: Font-Trebuchet MS, Size-9, Color-#287E8F, Display units-Auto, Title-Off
  • Legend-On, Position-Top left, Title-On
